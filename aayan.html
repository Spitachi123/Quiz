Direction of Vectors

The direction of a vector tells us which way the vector is pointing. Direction of the vector is important because two vectors with the same magnitude can be completely different if they point in different directions.

For example:
- 10 N force upwards
- 10 N force to the right
These produce different effects even though their magnitudes are equal.

Examples where direction is essential:
- Wind direction
- Movement of vehicles
- Navigation using maps
- Forces acting on objects
- Sports (e.g., throwing a ball)

Ways to Show the Direction of a Vector

The direction of a vector shows which way it points. There are several ways to represent it mathematically or descriptively.

1. Angle with a Reference Axis

A vector’s direction can be expressed as the angle it makes with a reference axis, usually the x-axis (horizontal axis).
This angle is measured in degrees or radians.
Example:
v⃗ = 10 units at 45 degree
Meaning: The vector has a magnitude of 10 units and points 45° above the horizontal x-axis, towards the top-right direction.
Using angles is useful when vectors are drawn or when calculating direction mathematically.

2. Vector Components

A vector can be split into horizontal (x) and vertical (y) components.
This method shows exactly how far the vector moves along each axis, which determines its direction.
Example:
v⃗ = 3i⃗ + 4j⃗
Meaning: The vector moves 3 units in the x-direction (right) and 4 units in the y-direction (up).
Using components makes it easy to calculate direction, especially for vectors not aligned along a single axis.

3. Compass Directions

Vectors can also be described using compass points, commonly used in navigation and practical applications.
Directions are expressed as North (N), South (S), East (E), West (W), or combinations like North-East (NE).
Example:
v⃗ = 5 m North-East
Meaning: The vector points diagonally between North and East.
Compass directions give an easy descriptive understanding of the vector’s orientation in space.

Types of Vectors

Row Vector
A vector written in a single horizontal row.
Example: v⃗ = [2,3,4]
Often used in matrices and linear algebra.

Column Vector
A vector written in a single vertical column.
Useful in matrix operations and computer programming.

Zero Vector (0⃗)
A vector with magnitude 0 and no specific direction.
Example: 0⃗ = [0,0,0]
Represents no movement or null effect.

Unit Vector (i^, j^, k^)
A vector with magnitude 1 used to indicate direction along axes.
Example: i^ = [1,0,0], j^ = [0,1,0], k^ = [0,0,1]
Unit vectors are often used to build other vectors using components.

Negative Vector (−A⃗)
A negative vector has the same magnitude as a given vector but points in the opposite direction.
It is represented as: −A⃗
Meaning: If A⃗ points right, −A⃗ points left; if A⃗ points up, −A⃗ points down.
Negative vectors are often used to reverse the effect of a vector, such as in forces, displacement, or velocity.
